Proyectos

Shopping List

Descripción: CRUD de un listado de mercancías realizado en Angular con un arreglo en TypeScript con doble enlace de datos, para su edición a partir del momento de la compra de cada producto hasta la posterior eliminación de los items que componen la lista.

Fecha de realización: Febrero 2022.

Poké Park

Descripción: En esta aplicación se consulta una API libre bastante conocida por miles de personas interesadas en aprender programación web, la PokéApi. Es una API JSON, que la hace más cómoda de manejar debido a que JSON
es un formato ligero de intercambio de datos de JavaScript.

Fecha de realización: Marzo 2022.

Mi Portfolio

Descripción: Portfolio personal desarrollado en arquitectura distribuída según el patrón de diseño de software Modelo Vista Controlador, con diseño de interfaz front end realizado con HTML5, CSS3, Bootstrap 5, JavaScript y Angular,
con MySQL como gestor de la base de datos integrada al desarrollo back end con Java y Spring Boot.

Fecha de realización: Mayo 2022.
