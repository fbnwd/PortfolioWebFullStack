
Caso Mi portfolio.

Creación de una aplicación SPA para el desarrollo de mi portfolio personal, realizado en arquitectura distribuída bajo el patrón de software MVC, que utiliza MySql como motor de gestión de datos. 
Para su desarrollo front end utiliza HTML5, CSS3, Bootstrap 5, JavaScript y su framework Angular12, y Java y su framework Spring boot, para su desarrollo back end.

El sitio contiene distintas secciones: Acerca de mí, Experiencia laboral, Educación, Habilidades y Proyectos.

Cada sección permite la realización del alta, edición y eliminación de datos, tarea a cargo del Administrador del sitio, quien es el responsable del inicio y cierre de sesión de usuario según la recuperación de su contraseña.

De acuerdo al contenido de cada sección, pueden utilizar distintos estilos y tamaños de textos para títulos, subtítulos y párrafos, logotipos en formatos de imagen .svg y .png, imágenes fotográficas en formato .jpg, animaciones hechas en JavaScript y transiciones hechas en CSS3 para optimizar su presentación.

Objetivo: Brindar información para la administración de las distintas secciones del sitio.

Límite: Desde la agregación de datos hasta su eliminación.
